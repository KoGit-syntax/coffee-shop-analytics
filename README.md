## Project Overview
This project features an interactive Excel dashboard designed to analyze six months of retail coffee shop sales data. The goal of this analysis was to transform raw transaction logs into actionable insights, identifying key revenue drivers and growth patterns to support informed business decision-making.

---

## 📊 Key Insights
*   *Total H1 Revenue:* $669,812
*   *Growth Trend:* Maintained a consistent upward trajectory from January through June 2023.
*   *Best-Performing Month:* June ($166,486).
*   *Peak Growth Period:* May 30th, achieving *31.8% Month-over-Month (MOM) growth* within the bracket.

---

## 🛠️ Skills Demonstrated
*   *Data Cleaning:* Transformed raw, unstructured transaction logs into a clean, analysis-ready format.
*   *Data Visualization:* Built dynamic, custom charts to illustrate revenue trends and comparative performance.
*   *Business Analytics:* Identified seasonal trends, demand spikes, and growth drivers to inform strategic planning.
